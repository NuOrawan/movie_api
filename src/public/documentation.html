myFlix Application

Documentation

myFlix Application is REST API that allow clients to access to information about different movies, directors, and genres via commonly used HTTP methods like GET, POST, PUT and DELETE. The application allows clients to update their information, for instance, their list of “Favorite Movies", usernames, passwords and emails.

URL Endpoints and Request Methods

Get all movies

URL

/movies

Method

GET

Request Body Format

None

Response Body Format

A JSON Object holding data about all movies.

Get description, genre, director, image URL of a single movie by title.

URL

/movies/[title]

Method

GET

Request Body Format

None

Response Body Format

A JSON object holding data about a single movie containing ID, title, description, genre, director and image. Example :
[
id : 1,
title : ‘Jurassic Park 2’
description : “is based on the 1990 novel written by Michael Crichton. “,
genre : {
name: “action”,
description : "fast-paced and include a lot of action like fight scenes, chase scenes."
}
director : “Steven Spielberg”,
birthyear : 1946,
month: 12,
date: 18,
bio: “Spielberge was born in Ohio and grew up in Arizona. He move to California and studied film in college."
]
